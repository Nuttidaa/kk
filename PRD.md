# Product Requirements Document (PRD)


Smart Class Check-in & Learning Reflection App
## Product Overview


Smart Class Check-in & Learning Reflection is a mobile and web application designed to help students check in to class and reflect on their learning after each session.



The application allows students to check attendance through QR Code or location verification and record what they learned during the class.



The application is developed using Flutter, allowing it to run on multiple platforms including mobile devices and web browsers such as Google Chrome.

## Problem Statement


In traditional classroom environments, attendance tracking and learning reflection are often managed manually, which can cause several issues:

Time-consuming attendance checking

Lack of student learning feedback

No centralized record of attendance history

Limited engagement from students after class



Therefore, a digital solution is needed to simplify attendance tracking and encourage student reflection.

## Product Goals


The main goals of this application are:

Simplify the class attendance process

Reduce time spent on manual attendance checking

Provide a digital record of attendance history

Encourage students to reflect on their learning

Improve interaction between students and instructors

## Core Features


5.1 Login System


Users can access the application through a login system.



Features include:

Email login

Password authentication

Optional guest access

5.2 Home Screen


The home screen serves as the main navigation page of the application.



Main menu options:

Check-in to Class

Finish Class

View History

5.3 Class Check-in


This feature allows students to check in when entering a class.



Information collected:

QR Code scan

GPS location verification

Previous class topic

Expected topic for today’s class

Student mood before class

5.4 Finish Class (Learning Reflection)


At the end of the class, students can submit their learning reflection.



Information collected:

What did you learn today?

Feedback about the class

5.5 Check-in History


This section displays the student’s attendance records.



Information displayed:

Course name

Date and time

Location

Attendance status
